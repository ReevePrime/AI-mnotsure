Most chatbots answer with confidence. This one answers, then immediately questions every life choice that led to that answer. 
AI-mnotsure is a RAG chatbot with a built-in eval suite that measures hallucination, faithfulness, and retrieval quality.

Because healthy self-doubt is a feature, not a bug!

Please note:
  Due to cost-limitations with using OpenAI for testing with Deepeval, the metrics have been generated using a local LLM (Qwen2.5:14b)
  The results might not be as accurate as a production version using OpenAI or Anthropic models.
  This repository is mostly presented as a portfolio project.
